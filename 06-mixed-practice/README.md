# Mixed JavaScript Practice

This section contains mixed JavaScript coding problems combining multiple concepts.

## Concepts Covered

- Array Methods (map, filter, reduce)
- Objects
- Object Destructuring
- Event Loop
- Async / Await
- Promises

## Problems Included

1. Filter users by age and return names
2. Nested object destructuring
3. Using reduce() to calculate sum
4. Event loop execution order
5. Async/Await with Promise
6. Filtering objects from an array

## Purpose

These exercises simulate real JavaScript interview questions where multiple concepts are tested together.
